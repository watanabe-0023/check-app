# 収益管理表 月次チェックアプリ

init株式会社の収益管理表と台帳・freee会計データを自動突合するStreamlitアプリ。

## 起動コマンド

```bash
cd check_app

# 依存パッケージインストール（初回のみ）
pip install -r requirements.txt

# アプリ起動
streamlit run app.py
```

ブラウザで `http://localhost:8501` が自動的に開きます。

## ファイル配置

実行ディレクトリに `.cache/` が自動作成され、月次ハッシュが保存されます。

```
check_app/
├── app.py
├── core/
├── config/
├── tests/
├── .cache/          ← 自動生成（I04: 前回差分検知用）
└── requirements.txt
```

## テスト実行

```bash
cd check_app
pytest tests/ -v
```

## 対応ファイル形式

| ファイル | 形式 |
|---|---|
| 収益管理表 | xlsx |
| 売上管理台帳 | xlsx |
| 支出管理表 | xlsx |
| 経費台帳 | xlsx（任意） |
| 事業別支出管理 | xlsx（任意） |
| freee 試算表(BS) | xlsx または PDF |
| freee 月次推移表(PL) | xlsx または PDF |

## 既知の構造的差異（info表示）

以下は差異が出るのが正常です。アプリ上では青色(参考)表示になります。

- **役員報酬/給与**: 管理表=振込ベース、freee=発生ベース
- **住民税/源泉所得税**: freeeはPL非計上（預り金）
- **労働保険料**: 管理表=月次積立、freee=6月一括
- **法人税等**: 管理表=毎月積立、freee=決算時のみ
